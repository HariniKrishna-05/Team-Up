# TEAM_UP Application

This Flask app manages student signup/login with OTP verification using an Oracle database.

## Setup

1. **Install dependencies**
   ```bash
   pip install -r requirement.txt
   ```

2. **Configure environment**
   - Copy `.env` to a local file and fill in the values:
     ```ini
     EMAIL_USER=your_email
     EMAIL_PASS=<your-gmail-app-password>
     SECRET_KEY=<random string>
     ORACLE_USER=your_username
     ORACLE_PASSWORD=your_password
     ORACLE_DSN=Your_dsn
     ```
   - Gmail requires an "app password" or allowing less-secure apps for SMTP.

3. **Initialize database**
   - The app expects two tables: `STUDENTS` and `OTP_VERIFICATION`.
   - Either navigate to <http://localhost:5000/init-db> once, or run the following SQL manually in your Oracle client:
     ```sql
     CREATE TABLE STUDENTS (
         STUDENT_ID NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
         NAME VARCHAR2(100),
         EMAIL VARCHAR2(255) UNIQUE,
         INSTITUTION VARCHAR2(200),
         PASSWORD VARCHAR2(255),
         IS_VERIFIED NUMBER(1) DEFAULT 0
     );

     CREATE TABLE OTP_VERIFICATION (
         OTP_ID NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
         STUDENT_ID NUMBER REFERENCES STUDENTS(STUDENT_ID),
         OTP VARCHAR2(6),
         EXPIRY_TIME DATE
     );
     ```
   - If you see `ORA-00942` errors during signup, the tables are missing; run initialization again.

4. **Run the application**
   ```bash
   python app.py
   ```

5. **Usage**
   - Sign up to create a new account. An OTP will be emailed to you.
   - If the OTP doesn't arrive, use the "Resend OTP" link.
   - Forgot password? Use the "Forgot password" link on the login page.

## Pages

- `/signup` – create a student account
- `/login` – authenticate verified users
- `/verify-otp` – enter OTP to verify email
- `/resend-otp` – generate a fresh OTP
- `/forgot-password` – begin password reset
- `/reset-password` – submit OTP + new password
- `/dashboard` – protected landing page
- `/logout` – clear session
- `/init-db` – initialize database tables (development)

## Troubleshooting

- **OTP not sending:** check `.env` credentials, Gmail app password, and server logs.
- **Database errors:** ensure Oracle is running and tables exist; use `/init-db`.
